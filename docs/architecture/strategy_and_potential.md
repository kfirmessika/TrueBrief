- **What**: An LLM-driven quality control gate.
- **The Achievement**: **Truth at Scale.** It doesn't just copy text; it *interprets* it. It rejects hallucinations, UI garbage, and errors, ensuring that only verified facts reach your dashboard.

### 3. The Memory (FactLedger)
- **What**: A local Vector Database (Qdrant).
- **The Achievement**: **Zero Redundancy.** Most news sites repeat the same stories for days. The Delta Engine "remembers" every fact it has ever seen. If it sees a fact it already knows, it ignores it. You only see the **Delta** (the change).

### 4. The Interface (The Stream)

- **The Achievement**: **Frictionless Consumption.** You go from 5,000 sources to 5 verified bullets on a single screen.
